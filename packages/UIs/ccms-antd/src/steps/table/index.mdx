# table模板
CCMS表单提交模板
常用的提交表单项 <br/>

## 使用方式
- (type:table) 配置静态数据展示
- (type:fetch) + (type:table)无条件直接查询
- (type:form) + (type:fetch) + (type:table)完成表单提交查询并展示
- (type:skip) + (type:fetch) + (type:table)完成自动查询并展示
- (type:filter) + (type:fetch) + (type:table)完成筛选条件查询并展示
