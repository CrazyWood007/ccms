# fetch模板
CCMS表单提交模板
常用的提交表单项 <br/>

## 使用方式
- (type:fetch) 接口调用
- (type:fetch) + (type:table)无条件直接查询
- (type:form) + (type:fetch)完成表单提交查询
- (type:skip) + (type:fetch)完成自动查询
- (type:filter) + (type:fetch)完成筛选条件查询
